# Duramen Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `duramen`, a Rust CLI tool that uses Cedar for fine-grained authorization of AI coding agent tool calls.

**Architecture:** A Cargo workspace with 6 crates: `cli` (binary), `engine` (Cedar integration + adapter trait), `normalizer` (agent-specific input normalization), `formatter` (agent-specific response formatting), `audit` (JSON line logging), and `policy-defaults` (embedded default policies). The `--agent` flag selects a matched normalizer+formatter pair.

**Tech Stack:** Rust, cedar-policy 4.x, clap 4.x (CLI), serde/serde_json (serialization), chrono (timestamps), uuid (request IDs), assert_cmd + predicates (integration tests)

```rust
// crates/engine/src/decision.rs
use serde::{Deserialize, Serialize};
use std::str::FromStr;

#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "kebab-case")]
pub enum DecisionTier {
    Allow,
    Audit,
    RequireApproval,
    Deny,
}

impl DecisionTier {
    pub fn exit_code(&self) -> i32 {
        match self {
            DecisionTier::Allow => 0,
            DecisionTier::Audit => 0,
            DecisionTier::RequireApproval => 2,
            DecisionTier::Deny => 1,
        }
    }
}

impl FromStr for DecisionTier {
    type Err = String;

    fn from_str(s: &str) -> Result<Self, Self::Err> {
        match s {
            "allow" => Ok(DecisionTier::Allow),
            "audit" => Ok(DecisionTier::Audit),
            "require-approval" => Ok(DecisionTier::RequireApproval),
            "deny" => Ok(DecisionTier::Deny),
            other => Err(format!("unknown decision tier: {other}")),
        }
    }
}

impl std::fmt::Display for DecisionTier {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            DecisionTier::Allow => write!(f, "allow"),
            DecisionTier::Audit => write!(f, "audit"),
            DecisionTier::RequireApproval => write!(f, "require-approval"),
            DecisionTier::Deny => write!(f, "deny"),
        }
    }
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AuthzDecision {
    pub decision: DecisionTier,
    pub reason: String,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub policy_id: Option<String>,
    pub evaluation_time_ms: u64,
}

impl AuthzDecision {
    pub fn new(decision: DecisionTier, reason: String) -> Self {
        Self {
            decision,
            reason,
            policy_id: None,
            evaluation_time_ms: 0,
        }
    }

    pub fn is_allowed(&self) -> bool {
        matches!(self.decision, DecisionTier::Allow | DecisionTier::Audit)
    }

    pub fn exit_code(&self) -> i32 {
        self.decision.exit_code()
    }
}
```

```rust
}

impl AgentPrincipal {
    pub fn new(agent_type: &str) -> Self {
        Self {
            agent_type: agent_type.to_string(),
            session_id: None,
            user: None,
        }
    }
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AuthzAction {
    pub name: String,
}

impl AuthzAction {
    pub fn new(name: &str) -> Self {
        Self {
            name: name.to_string(),
        }
    }
}

#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum ResourceType {
    File,
    Command,
    Url,
    GitRef,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AuthzResource {
    pub resource_type: ResourceType,
    pub id: String,
    #[serde(default, skip_serializing_if = "serde_json::Value::is_null")]
    pub attributes: serde_json::Value,
}

impl AuthzResource {
    pub fn file(path: &str) -> Self {
        Self {
            resource_type: ResourceType::File,
            id: path.to_string(),
            attributes: serde_json::Value::Null,
        }
    }

    pub fn command(cmd: &str) -> Self {
        Self {
            resource_type: ResourceType::Command,
            id: cmd.to_string(),
            attributes: serde_json::Value::Null,
        }
    }

    pub fn url(url: &str) -> Self {
        Self {
            resource_type: ResourceType::Url,
            id: url.to_string(),
            attributes: serde_json::Value::Null,
        }
    }

    pub fn git_ref(ref_name: &str) -> Self {
        Self {
            resource_type: ResourceType::GitRef,
            id: ref_name.to_string(),
            attributes: serde_json::Value::Null,
        }
    }
}
```

- [ ] **Step 4: Write audit-file-writes.cedar**

Note: Cedar does not have a native "audit" decision. The `audit` tier is implemented in the engine layer: if a request is `Allow` by Cedar and matches an audit annotation, the engine promotes it to `DecisionTier::Audit`. For now, we use a permit policy for file writes (the engine handles audit logic separately).

```cedar
// policies/default/audit-file-writes.cedar
// Allow file creation and editing on non-protected resources.
// The engine promotes these to "audit" tier for logging.
// This is annotated with @advice("audit") for the engine to detect.
@advice("audit")
permit(
    principal,
    action in [
        Action::"file:create",
        Action::"file:edit"
    ],
    resource
) when {
    resource.is_protected != true
};
```

```rust
// crates/engine/src/policy.rs
use crate::adapter::EngineError;
use std::path::Path;

pub struct PolicyLoader;

impl PolicyLoader {
    pub fn new() -> Self {
        Self
    }

    /// Load all .cedar files from a directory and return their contents.
    pub fn load_from_dir(&self, dir: &Path) -> Result<Vec<String>, EngineError> {
        if !dir.exists() {
            return Ok(Vec::new());
        }

        let mut policies = Vec::new();
        for entry in std::fs::read_dir(dir).map_err(EngineError::Io)? {
            let entry = entry.map_err(EngineError::Io)?;
            let path = entry.path();
            if path.extension().is_some_and(|ext| ext == "cedar") {
                let content = std::fs::read_to_string(&path).map_err(EngineError::Io)?;
                // Validate it parses
                content.parse::<cedar_policy::PolicySet>().map_err(|e| {
                    EngineError::PolicyParse(format!("{}: {e}", path.display()))
                })?;
                policies.push(content);
            }
        }

        Ok(policies)
    }

    /// Load policies with resolution order: repo dir > user dir > defaults.
    pub fn load_merged(
        &self,
        repo_dir: Option<&Path>,
        user_dir: Option<&Path>,
        defaults: &[&str],
    ) -> Result<cedar_policy::PolicySet, EngineError> {
        let mut all_sources: Vec<String> = Vec::new();

        // Defaults first (lowest priority, but Cedar uses deny-overrides so order matters less)
        for d in defaults {
            all_sources.push(d.to_string());
        }

        // User-level policies
        if let Some(dir) = user_dir {
            all_sources.extend(self.load_from_dir(dir)?);
        }

        // Repo-level policies (highest priority)
        if let Some(dir) = repo_dir {
            all_sources.extend(self.load_from_dir(dir)?);
        }

        let combined = all_sources.join("\n");
        combined
            .parse::<cedar_policy::PolicySet>()
            .map_err(|e| EngineError::PolicyParse(format!("{e}")))
    }
}
```

```rust
// crates/engine/src/evaluator.rs
use crate::adapter::{EngineError, PolicyEngine};
use crate::decision::{AuthzDecision, DecisionTier};
use crate::entities::AuthzRequest;
use cedar_policy::{Authorizer, Context, Entities, PolicySet, Request};
use std::time::Instant;

pub struct CedarEngine {
    policy_set: PolicySet,
}

impl CedarEngine {
    pub fn from_policy_str(policy_src: &str) -> Result<Self, EngineError> {
        let policy_set: PolicySet = if policy_src.trim().is_empty() {
            PolicySet::new()
        } else {
            policy_src
                .parse()
                .map_err(|e| EngineError::PolicyParse(format!("{e}")))?
        };
        Ok(Self { policy_set })
    }

    pub fn from_policy_set(policy_set: PolicySet) -> Self {
        Self { policy_set }
    }

    fn build_request(
        &self,
        request: &AuthzRequest,
    ) -> Result<Request, EngineError> {
        let principal = format!(
            r#"Agent::"{}""#,
            request.principal.agent_type
        )
        .parse()
        .map_err(|e| EngineError::Evaluation(format!("bad principal: {e}")))?;

        let action = format!(
            r#"Action::"{}""#,
            request.action.name
        )
        .parse()
        .map_err(|e| EngineError::Evaluation(format!("bad action: {e}")))?;

        let resource_type = match request.resource.resource_type {
            crate::entities::ResourceType::File => "File",
            crate::entities::ResourceType::Command => "Command",
            crate::entities::ResourceType::Url => "Url",
            crate::entities::ResourceType::GitRef => "GitRef",
        };
        let resource = format!(
            r#"{}::"{}""#,
            resource_type, request.resource.id
        )
        .parse()
        .map_err(|e| EngineError::Evaluation(format!("bad resource: {e}")))?;

        let context = Context::empty();

        Request::new(principal, action, resource, context, None)
            .map_err(|e| EngineError::Evaluation(format!("request build error: {e}")))
    }

    fn check_advice(&self, response: &cedar_policy::Response) -> Option<DecisionTier> {
        for reason_id in response.diagnostics().reason() {
            if let Some(policy) = self.policy_set.policy(&reason_id) {
                if let Some(annotation) = policy.annotation("advice") {
                    return match annotation.as_ref() {
                        "audit" => Some(DecisionTier::Audit),
                        "require-approval" => Some(DecisionTier::RequireApproval),
                        _ => None,
                    };
                }
            }
        }
        None
    }
}

impl PolicyEngine for CedarEngine {
    fn evaluate(&self, request: &AuthzRequest) -> Result<AuthzDecision, EngineError> {
        let start = Instant::now();
        let cedar_request = self.build_request(request)?;
        let authorizer = Authorizer::new();
        let entities = Entities::empty();

        let response = authorizer.is_authorized(&cedar_request, &self.policy_set, &entities);
        let elapsed = start.elapsed().as_millis() as u64;

        let base_decision = match response.decision() {
            cedar_policy::Decision::Allow => DecisionTier::Allow,
            cedar_policy::Decision::Deny => DecisionTier::Deny,
        };

        // Check for advice annotations that promote Allow to Audit/RequireApproval
        let final_decision = if base_decision == DecisionTier::Allow {
            self.check_advice(&response).unwrap_or(DecisionTier::Allow)
        } else {
            base_decision
        };

        let reason = match final_decision {
            DecisionTier::Allow => "request allowed by policy".to_string(),
            DecisionTier::Audit => "request allowed (audit logged)".to_string(),
            DecisionTier::RequireApproval => "request requires human approval".to_string(),
            DecisionTier::Deny => {
                let errors: Vec<String> = response
                    .diagnostics()
                    .errors()
                    .map(|e| e.to_string())
                    .collect();
                if errors.is_empty() {
                    "request denied by policy".to_string()
                } else {
                    format!("request denied: {}", errors.join("; "))
                }
            }
        };

        // Collect the policy ID from the first reason
        let policy_id = response
            .diagnostics()
            .reason()
            .next()
            .map(|id| id.to_string());

        Ok(AuthzDecision {
            decision: final_decision,
            reason,
            policy_id,
            evaluation_time_ms: elapsed,
        })
    }

    fn validate_policies(&self) -> Result<(), EngineError> {
        // Policies were already parsed during construction;
        // this is a no-op for CedarEngine but exists for the trait.
        Ok(())
    }
}
```

```rust
use crate::traits::{AgentNormalizer, NormalizerError};
use duramen_engine::entities::*;

pub struct CopilotCliNormalizer;

impl CopilotCliNormalizer {
    fn tool_to_action(tool: &str) -> &str {
        match tool {
            "view" => "file:read",
            "edit" => "file:edit",
            "create" => "file:create",
            "powershell" | "bash" => "shell:execute",
            "grep" | "glob" => "file:read",
            "web_fetch" => "network:fetch",
            _ => "tool:unknown",
        }
    }

    fn tool_to_resource(tool: &str, args: &serde_json::Value) -> AuthzResource {
        match tool {
            "view" | "edit" | "create" => {
                let path = args.get("path")
                    .and_then(|v| v.as_str())
                    .unwrap_or("unknown");
                AuthzResource::file(path)
            }
            "powershell" | "bash" => {
                let cmd = args.get("command")
                    .and_then(|v| v.as_str())
                    .unwrap_or("unknown");
                let is_destructive = Self::is_destructive_command(cmd);
                let mut r = AuthzResource::command(cmd);
                r.attributes = serde_json::json!({"is_destructive": is_destructive});
                r
            }
            "web_fetch" => {
                let url = args.get("url")
                    .and_then(|v| v.as_str())
                    .unwrap_or("unknown");
                AuthzResource::url(url)
            }
            _ => AuthzResource::command("unknown"),
        }
    }

    fn is_destructive_command(cmd: &str) -> bool {
        let lower = cmd.to_lowercase();
        lower.contains("rm -rf")
            || lower.contains("sudo ")
            || lower.contains("format ")
            || lower.starts_with("del ")
            || lower.contains("remove-item")
            || lower.contains("git push --force")
            || lower.contains("git push -f")
    }
}
```

```rust
use duramen_engine::decision::DecisionTier;
```

```rust
use chrono::Utc;
use serde::Serialize;
use std::fs::{self, OpenOptions};
use std::io::Write;

use uuid::Uuid;

#[derive(Debug, Serialize)]
pub struct AuditEntry {
    pub timestamp: String,
    pub request_id: String,
    pub principal: AuditPrincipal,
    pub action: String,
    pub resource: AuditResource,
    pub context: serde_json::Value,
    pub raw_command: serde_json::Value,
    pub decision: DecisionTier,
    pub reason: String,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub policy_id: Option<String>,
    pub evaluation_time_ms: u64,
}

#[derive(Debug, Serialize)]
pub struct AuditPrincipal {
    #[serde(rename = "type")]
    pub principal_type: String,
    pub id: String,
}

#[derive(Debug, Serialize)]
pub struct AuditResource {
    #[serde(rename = "type")]
    pub resource_type: String,
    pub path: String,
}

impl AuditEntry {
    pub fn new(
        request: &AuthzRequest,
        decision: DecisionTier,
        reason: String,
        policy_id: Option<String>,
        evaluation_time_ms: u64,
        raw_command: serde_json::Value,
    ) -> Self {
        Self {
            timestamp: Utc::now().to_rfc3339(),
            request_id: Uuid::new_v4().to_string(),
            principal: AuditPrincipal {
                principal_type: "Agent".into(),
                id: request.principal.agent_type.clone(),
            },
            action: request.action.name.clone(),
            resource: AuditResource {
                resource_type: format!("{:?}", request.resource.resource_type),
                path: request.resource.id.clone(),
            },
            context: serde_json::json!({
                "tool": request.context.tool_name,
                "working_dir": request.context.working_directory,
            }),
            raw_command,
            decision,
            reason,
            policy_id,
            evaluation_time_ms,
        }
    }
}

pub struct AuditLogger {
    log_path: PathBuf,
}

impl AuditLogger {
    pub fn new(log_path: &Path) -> Result<Self, std::io::Error> {
        if let Some(parent) = log_path.parent() {
            fs::create_dir_all(parent)?;
        }
        Ok(Self {
            log_path: log_path.to_path_buf(),
        })
    }

    pub fn log(&self, entry: &AuditEntry) -> Result<(), std::io::Error> {
        let json = serde_json::to_string(entry)
            .map_err(|e| std::io::Error::new(std::io::ErrorKind::Other, e))?;

        let mut file = OpenOptions::new()
            .create(true)
            .append(true)
            .open(&self.log_path)?;
        writeln!(file, "{json}")?;
        Ok(())
    }

    pub fn log_path(&self) -> &Path {
        &self.log_path
    }
}
```

```rust
// crates/cli/src/commands/check.rs
use duramen_audit::logger::{AuditEntry, AuditLogger};
use duramen_engine::adapter::PolicyEngine;
use duramen_engine::decision::DecisionTier;
use duramen_engine::entities::*;
use duramen_engine::evaluator::CedarEngine;
use duramen_engine::policy::PolicyLoader;
use duramen_request_adaptor::get_normalizer;
use duramen_response_formatter::get_formatter;
use std::io::Read;
use std::path::PathBuf;

pub fn run(
    agent: Option<String>,
    principal: Option<String>,
    action: Option<String>,
    resource: Option<String>,
    resource_type: String,
    context: Option<String>,
    policy_dir: Option<String>,
    audit_log: Option<String>,
) -> i32 {
    let result = run_inner(agent, principal, action, resource, resource_type, context, policy_dir, audit_log);
    match result {
        Ok(code) => code,
        Err(e) => {
            let error_json = serde_json::json!({"error": e.to_string()});
            eprintln!("{}", serde_json::to_string(&error_json).unwrap());
            3
        }
    }
}

fn run_inner(
    agent: Option<String>,
    principal: Option<String>,
    action: Option<String>,
    resource: Option<String>,
    resource_type: String,
    context: Option<String>,
    policy_dir: Option<String>,
    audit_log: Option<String>,
) -> Result<i32, Box<dyn std::error::Error>> {
    let agent_name = agent.as_deref().unwrap_or("generic");

    // Build the AuthzRequest either from stdin (agent mode) or explicit args
    let (request, raw_command) = if agent.is_some() && principal.is_none() {
        // Read from stdin
        let mut input = String::new();
        std::io::stdin().read_to_string(&mut input)?;
        let payload: RawHookPayload = serde_json::from_str(&input)?;
        let raw_cmd = serde_json::json!({"tool": payload.tool, "args": payload.args});
        let normalizer = get_normalizer(agent_name)?;
        let req = normalizer.normalize(&payload)?;
        (req, raw_cmd)
    } else {
        // Build from explicit args
        let principal_name = principal.ok_or("--principal is required when not using --agent with stdin")?;
        let action_name = action.ok_or("--action is required")?;
        let resource_id = resource.ok_or("--resource is required")?;

        let res = match resource_type.as_str() {
            "file" => AuthzResource::file(&resource_id),
            "command" => AuthzResource::command(&resource_id),
            "url" => AuthzResource::url(&resource_id),
            "gitref" => AuthzResource::git_ref(&resource_id),
            other => return Err(format!("unknown resource type: {other}").into()),
        };

        let ctx_json: serde_json::Value = context
            .as_deref()
            .map(serde_json::from_str)
            .transpose()?
            .unwrap_or(serde_json::json!({}));

        let tool_name = ctx_json.get("tool")
            .and_then(|v| v.as_str())
            .unwrap_or("unknown")
            .to_string();

        let req = AuthzRequest {
            principal: AgentPrincipal::new(&principal_name),
            action: AuthzAction::new(&action_name),
            resource: res,
            context: AuthzContext {
                tool_name,
                working_directory: ctx_json.get("working_dir")
                    .and_then(|v| v.as_str())
                    .map(String::from),
                extra: ctx_json.clone(),
            },
        };
        let raw_cmd = ctx_json;
        (req, raw_cmd)
    };

    // Load policies
    let loader = PolicyLoader::new();
    let defaults = duramen_policy_defaults::all_default_policies();
    let default_refs: Vec<&str> = defaults.iter().map(|s| *s).collect();

    let repo_dir = policy_dir.map(PathBuf::from);
    let user_dir = dirs_next::config_dir().map(|d| d.join("duramen"));

    let policy_set = loader.load_merged(
        repo_dir.as_deref(),
        user_dir.as_deref(),
        &default_refs,
    )?;

    // Evaluate
    let engine = CedarEngine::from_policy_set(policy_set);
    let decision = engine.evaluate(&request)?;

    // Audit log if decision is Audit or Deny
    if matches!(decision.decision, DecisionTier::Audit | DecisionTier::Deny | DecisionTier::RequireApproval) {
        let log_path = audit_log
            .map(PathBuf::from)
            .unwrap_or_else(|| {
                dirs_next::home_dir()
                    .unwrap_or_else(|| PathBuf::from("."))
                    .join(".duramen")
                    .join("audit.log")
            });

        if let Ok(logger) = AuditLogger::new(&log_path) {
            let entry = AuditEntry::new(
                &request,
                decision.decision,
                decision.reason.clone(),
                decision.policy_id.clone(),
                decision.evaluation_time_ms,
                raw_command,
            );
            let _ = logger.log(&entry);
        }
    }

    // Format response
    let formatter = get_formatter(agent_name);
    let response = formatter.format(&decision, &request);
    println!("{}", response.stdout);

    Ok(response.exit_code)
}
```

```rust
        ("require-approval-sensitive.cedar", duramen_policy_defaults::REQUIRE_APPROVAL_SENSITIVE),

    for (name, content) in &defaults {

        if let Err(e) = fs::write(&path, content) {
            eprintln!(r#"{{"error":"failed to write {}: {}"}}"#, name, e);
            return 3;
        }
    }

    println!(r#"{{"status":"initialized","files_created":{}}}"#, defaults.len());
```

# Duramen

Fine-grained authorization for AI coding agents using [Cedar](https://www.cedarpolicy.com/).

Duramen intercepts tool calls from coding agents (Copilot CLI, Cursor, Codex, etc.) and evaluates them against Cedar policies before execution. It enforces a four-tier decision model: **allow**, **audit** (allow + log), **require-approval** (block + prompt user), and **deny**.

## Architecture

Rust Cargo workspace with 6 crates:

- **engine** (`crates/engine/`) — Core authorization: `PolicyEngine` trait, `CedarEngine` implementation, entity types (`AuthzRequest`, `AuthzDecision`), policy loading. All other crates depend on this.
- **normalizer** (`crates/request-adaptor/`) — Converts agent-specific hook payloads into `AuthzRequest`. `AgentNormalizer` trait with `CopilotCliNormalizer` and `GenericNormalizer`.
- **formatter** (`crates/response-formatter/`) — Converts `AuthzDecision` into agent-specific response format. `ResponseFormatter` trait. Paired with normalizers via `--agent` flag.
- **audit** (`crates/audit/`) — JSON-line structured audit logging with `AuditLogger`.
- **policy-defaults** (`crates/policy-defaults/`) — Embeds default Cedar policies at compile time.
- **cli** (`crates/cli/`) — Binary entry point. Subcommands: `check`, `validate`, `init`.

## Key Conventions

- **Adapter pair pattern:** The `--agent` flag selects both a normalizer (input) and formatter (output) as a matched pair. Adding a new agent = implement `AgentNormalizer` + `ResponseFormatter` + register in both `request-adaptor/src/lib.rs` and `response-formatter/src/lib.rs`.
- **Fail-closed:** Any engine error results in a `deny` decision. Security systems never fail-open.
- **Decision tiers:** `allow` (exit 0), `audit` (exit 0 + log), `require-approval` (exit 2), `deny` (exit 1), `error` (exit 3).
- **Cedar advice annotations:** `@advice("audit")` and `@advice("require-approval")` on Cedar permit policies promote Allow decisions to Audit/RequireApproval tiers. See `evaluator.rs::check_advice()`.
- **Policy resolution:** repo `.authz/` > user `~/.config/duramen/` > compiled-in defaults.
- **Entity model:** Principals are `Agent::`, resources are `File::`, `Command::`, `Url::`, `GitRef::`. All under the `duramen` Cedar namespace.
